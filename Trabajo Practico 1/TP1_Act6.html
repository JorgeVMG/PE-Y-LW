INTRODUCCION HTML


Este es un curso que pretende introducir al lector en el mundo del
HTML. No se requieren conocimientos previos, se empezara desde cero.
No hay que saber nada de programacion para utilizar HTML, basta
aprender un par de cosas sencillas para empezar a crear paginas.

Es MUY FACIL aprender HTML. Tan solo necesitas un editor de textos, un
curso de HTML y un navegador web (como netscape o internet explorer),
ni siquiera necesitas estar conectado a Internet.
